# Bakers Choice – Desktop Application

Bakers Choice is a desktop bakery e-commerce and learning application designed to allow users to browse bakery products, add items to cart, and learn baking through tips and content.

## Project Objective
The goal of this project is to design and build a scalable, user-friendly desktop application with a clean UI and reusable components, following a UI-first development approach.

## Design Phase
- Initial layouts sketched on paper
- Wireframes created and finalized in Figma
- Common layout structure with header, sidebar, and footer
- Product listing page designed with pagination (6 items per page)
- Navigation flow prototyped between screens

## Pages Designed
- Home (After Login)
- Products
- Contact Us
- About Us

## Tools Used
- Figma (Wireframes & Prototyping)
- GitHub (Version control & documentation)
